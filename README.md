# Aura Elysian 🛒
Full-stack E-commerce platform built with Node.js, Express, MongoDB and TypeScript.

## 🚀 Features
- Product catalog and product posting system
- Employee dashboard for managing products and orders
- Secure REST APIs
- Payment integration
- Comment and user interaction system
- Responsive UI for desktop and mobile

## 🛠 Tech Stack
- TypeScript
- Node.js
- Express.js
- MongoDB
- Ubuntu (VPS)
- REST APIs
